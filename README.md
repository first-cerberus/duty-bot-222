# DutyBOT

Telegram bot для управління єбашками

## Встановлення

1. Встановіть залежності:
```bash
npm install
```

2. Створіть файл `.env` на основі `.env.example` та додайте ваш токен бота

3. Переконайтесь що MongoDB запущено

4. Запустіть бота:
```bash
npm start
```

Для режиму розробки:
```bash
npm run dev
```

## Команди бота

- `/start_duty` - Запуск меню управління чергами
